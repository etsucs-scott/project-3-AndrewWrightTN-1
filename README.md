# CSCI 1260 — Project

INTRUCTIONS ----------------------------------------------------

Run the game.

You have a menu popup.

The user now has a choice of 1, 2, or 3

1- Is a 8x8 grid board

2- Is a 12x12 grid board

3- Is a 16x16 grid board


After selecting your board you're asked to input you're seed.

You're seed is related to your board, if you play again with the same seed, it will be the same board.

After you chose a seed (or left it blank) you're board is loaded.

Now you have a few commands to play with:

r *row* *column* Ex. r 4 6    - Reveals the forth row and the 5 column

f *row* *column* Ex. r 4 6    - Flags the forth row and the 5 column

q - will quit
